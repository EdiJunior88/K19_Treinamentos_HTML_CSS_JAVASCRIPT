K02 - Desenvolvimento Web com HTML, CSS e JavaScrip

Atualmente, praticamente todos os sistemas corporativos posseum interfaces web. Para quem deseja atuar no mercado de desenvolvimento de software, é obrigatório o conhecimento das linguagens: HTML, CSS e JavaScript.

Essas linguagens são utilizadas para o desenvolvimento da camada de apresentação das aplicações web.

Pré-requisitos

  • Familiaridade com algum sistema operacional (Windows / Linux / Mac OS X)
  • Lógica de Programação
